# Play Store 등록정보

## 앱 이름
촌맵 - 우리 가족 가계도

## 간단한 설명 (80자 이내)
한국 촌수 기반 가계도 앱. 가족을 탭하면 80가지 호칭이 자동으로 바뀝니다.

## 자세한 설명 (4000자 이내)
촌맵은 한국 촌수 체계를 기반으로 한 가계도 앱입니다.

가족을 추가하고 탭하면, 그 사람 기준으로 모든 가족의 호칭과 촌수가 자동으로 계산됩니다. "아버지의 형은 뭐라고 부르지?" 더 이상 고민하지 마세요.

◆ 핵심 기능
• 8촌까지 80가지 이상의 한국식 호칭 자동 계산
• 부계/모계/인척 구분 (장인어른, 시어머니, 처남, 형수 등)
• 인물을 탭하면 기준 변경 → 모든 호칭 즉시 재계산
• 장유 구분 (형/동생, 큰아버지/작은아버지)
• 외손자/친손자 자동 구분

◆ 가계도 시각화
• 세대별 트리 레이아웃
• 브래킷 스타일 연결선
• 혈연(실선) / 배우자(점선) 구분
• 경로 하이라이트 (선택한 사람까지의 관계 경로)
• 세대 라벨 표시
• 핀치 줌, 드래그 팬 지원

◆ 관계 검색
• 두 사람 선택 → 호칭 + 촌수 + 경로 표시
• 촌수별 필터 검색 ("나의 4촌은 누구?")
• 다중 경로 표시 (여러 관계 경로가 있을 때)
• 양방향 호칭 동시 표시

◆ 편의 기능
• 프로필 사진 설정
• 고인 표시 (사망년도 + 생몰년)
• 메모 기능 (기일, 연락처 등)
• 가족 템플릿 3종 (핵가족, 3대, 양가)
• Undo/Redo (20단계)
• 이미지로 저장 (PNG)
• 데이터 백업/복원 (JSON)
• 클립보드 공유
• 다크 모드
• 오프라인 지원

◆ 이런 분께 추천합니다
• 명절마다 친척 호칭이 헷갈리는 분
• 가계도를 정리하고 싶은 분
• 아이에게 가족 관계를 알려주고 싶은 부모님
• 촌수 계산이 궁금한 모든 분

모든 데이터는 기기에만 저장되어 개인정보가 외부로 전송되지 않습니다.
