촌수 계산법 완전 정리

1촌부터 8촌까지 실제 사례로 익히는 촌수 계산

촌수는 두 사람을 잇는 가계도 위 선의 개수로 계산합니다. 규칙은 간단하지만, 부부(배우자) 사이에는 촌수가 없다는 점과 부모-자식 관계를 건너는 것만 한 단계로 친다는 점을 기억하면 누구나 쉽게 계산할 수 있습니다.

기본 계산 규칙 3가지

  1. 부모-자식 간 이동은 1촌: 한 세대를 위 또는 아래로 넘어가면 1촌이 더해집니다.
  2. 배우자 간 이동은 0촌: 부부는 하나로 봅니다. 형수를 계산할 때 "형(2촌) → 형수(2촌)"이 되는 이유입니다.
  3. 공통 조상까지 올라갔다가 내려온다: 직계가 아닌 관계는 공통 조상을 찾아 "올라간 칸 + 내려온 칸"으로 셉니다.

1촌 - 부모와 자녀

가장 가까운 혈연 관계입니다. 한 세대 차이로 부모-자식 관계를 이룹니다.

관계호칭경로
아버지 · 어머니아버지 / 어머니나 → 부모 (1단계)
아들 · 딸아들 / 딸나 → 자녀 (1단계)

2촌 - 형제자매, 조부모, 손자녀

세대가 같은 형제자매와, 두 세대 차이인 조부모·손자녀가 모두 2촌입니다.

관계호칭경로
형제자매형, 누나, 오빠, 언니, 동생나 → 부모 → 형제자매
조부모할아버지, 할머니, 외할아버지, 외할머니나 → 부모 → 조부모
손자녀손자, 손녀나 → 자녀 → 손자녀

여기서 외조부모는 어머니의 부모입니다. 촌수는 같지만 부계·모계에 따라 호칭이 달라집니다.

3촌 - 삼촌, 이모, 조카

부모의 형제자매, 또는 형제자매의 자녀입니다.

관계호칭경로
아버지의 형큰아버지(백부)나 → 아버지 → 할아버지 → 큰아버지
아버지의 남동생작은아버지(숙부)나 → 아버지 → 할아버지 → 작은아버지
아버지의 자매고모나 → 아버지 → 할아버지 → 고모
어머니의 남형제외삼촌나 → 어머니 → 외할아버지 → 외삼촌
어머니의 자매이모나 → 어머니 → 외할아버지 → 이모
형제자매의 자녀조카, 조카딸나 → 부모 → 형제자매 → 조카

공통 조상(조부모)까지 2단계 올라갔다가 1단계 내려오므로 2 + 1 = 3촌이 됩니다.

4촌 - 사촌, 종조부(큰할아버지), 증조부모

사촌은 부모의 형제자매의 자녀, 즉 같은 조부모를 둔 사이입니다.

관계호칭경로
아버지 쪽 사촌사촌(형/누나/동생)나 → 아버지 → 할아버지 → 큰·작은아버지 → 사촌
어머니 쪽 사촌외사촌나 → 어머니 → 외할아버지 → 외삼촌 → 외사촌
할아버지의 형큰할아버지(종조부)나 → 아버지 → 할아버지 → 증조할아버지 → 큰할아버지
증조부모증조할아버지, 증조할머니나 → 부 → 조 → 증조(3단계)

사촌은 조부모까지 2단계 올라가 삼촌 거쳐 2단계 내려오니 2 + 2 = 4촌입니다.

5촌 - 당숙, 5촌 조카

사촌의 자녀, 또는 부모의 사촌에 해당합니다. 같은 세대에서는 나오지 않고 한 세대 위·아래로만 존재합니다.

관계호칭경로
아버지의 사촌(남성)당숙 (오촌 아저씨)공통 조상: 증조부
아버지의 사촌(여성)당고모공통 조상: 증조부
사촌의 자녀5촌 조카공통 조상: 할아버지

6촌 - 육촌

증조부모를 공통 조상으로 하는 사이입니다. 내 부모와 사촌이라면, 그 자녀와 나는 6촌이 됩니다.

관계호칭공통 조상
6촌 (당질, 재종간)육촌 형제, 육촌 누이, 재종형제증조부모

경로: 나 → 부 → 조 → 증조 → 당숙 → 육촌 (3단계 상승 + 3단계 하강 = 6촌)

7촌 - 재종숙, 재종조카

고조부모를 공통 조상으로 하며, 세대가 하나 어긋납니다.

관계호칭
재종숙(7촌 위)재종숙
재종조카(7촌 아래)재종조카, 재종질녀

8촌 - 팔촌

민법상 친족의 경계선입니다. 고조부모를 공통 조상으로 하는 같은 세대입니다. 한 집안의 가장 바깥쪽 친족으로 여겨집니다.

관계호칭
8촌 형제자매팔촌 형/누나/동생 (삼종형제)

경로: 나 → 부 → 조 → 증조 → 고조 → 고조의 다른 자녀 → 그 자녀 → 그 자녀 → 8촌 (4단계 상승 + 4단계 하강)

헷갈리기 쉬운 사례

1. 형수는 몇 촌인가요?

형과 형수는 부부(무촌)이므로, 형수는 나와 2촌 관계인 인척입니다. 촌수는 형과 같지만 호칭은 "형수"로 다릅니다.

2. 사돈은 촌수가 있나요?

사돈은 친족 체계 바깥입니다. 형제자매의 배우자의 부모 등은 엄밀히 말하면 친족 범위를 벗어난 사회적 관계로, 촌수를 계산하지 않습니다.

3. 배우자의 삼촌은 몇 촌?

배우자는 무촌이므로 건너뛰고, 배우자의 3촌이 곧 내 3촌 인척이 됩니다. 다만 호칭은 혈족 3촌과 다릅니다(예: 배우자의 외삼촌은 "처외삼촌").
