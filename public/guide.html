촌맵 이용 가이드

5분이면 익히는 한국식 가계도 만들기

촌맵은 별도의 회원가입이나 설치 없이 바로 사용할 수 있습니다. 이 가이드를 따라 하시면 5~10분 안에 우리 가족 가계도를 완성하고, 복잡한 친척 호칭까지 한눈에 파악할 수 있습니다.

↑ 기본 3대 가족 가계도 예시 (본인 기준)

1단계 — 시작하기

촌맵 웹사이트(chonmap.vercel.app)에 접속하면 가운데에 "나" 노드가 하나 있는 빈 가계도가 표시됩니다. 처음 방문하시는 분을 위한 간단한 온보딩 안내가 나타나며, "시작하기" 버튼을 누르면 바로 가계도를 편집할 수 있습니다.

템플릿으로 빠르게 시작하기

직접 하나씩 추가하기 번거롭다면 설정 > 가족 템플릿에서 세 가지 템플릿 중 하나를 선택해 시작할 수 있습니다.

  • 핵가족 4인: 부모 + 자녀 2명의 기본 구성
  • 3대 가족 8인: 조부모까지 포함한 확장 구성
  • 양가 가족 12인: 친가·외가 양쪽을 포함한 풀 구성

2단계 — 가족 추가하기

화면 우측 하단의 ➕(플러스) 버튼을 누르면 누구를 추가할지 선택할 수 있습니다. 선택할 수 있는 관계는 다음과 같습니다.

  • 아버지 / 어머니 (부모 추가)
  • 아들 / 딸 (자녀 추가)
  • 배우자 (1인만 허용)
  • 형 / 동생 / 누나 / 여동생 (형제자매)

관계를 선택하면 이름, 성별, 출생년도, 생일(선택), 메모(선택) 등을 입력하는 바텀시트가 나타납니다. 필수 항목은 이름뿐이며, 나머지는 나중에 채워도 됩니다.

💡 팁: 출생년도를 입력하면 촌맵이 "큰아버지"와 "작은아버지"를 자동으로 구분해 줍니다. 형제 중 누가 연상인지도 알아서 판단합니다.

3단계 — 가족 정보 수정·삭제

가족 구성원을 길게 누르면(또는 PC에서 클릭 후 하단 FAB의 ✎ 버튼) 수정·삭제 바텀시트가 나타납니다. 여기서 다음을 할 수 있습니다.

  • 이름·성별·출생년도·사망년도 수정
  • 생일·기일 입력(MM-DD 형식)
  • 메모 추가
  • 프로필 사진 업로드(자동 압축, 500KB 제한)
  • 구성원 삭제(루트 인물 "나"는 삭제 불가)

부모를 삭제하면 다른 부모가 없는 자녀는 함께 삭제됩니다. 실수한 경우 Ctrl+Z 또는 헤더의 "되돌리기" 버튼으로 복구할 수 있습니다(최대 20단계).

4단계 — 기준 인물 바꾸기 (촌맵의 핵심 기능)

촌맵의 가장 큰 특징은 사람을 탭하면 그 사람이 기준이 된다는 점입니다.

본인 기준 화면

홍길동 기준으로 변경 시

위 두 화면을 비교해 보세요. 기준이 바뀌면 호칭이 모두 재계산되어, 같은 사람이 "아들"에서 "본인"으로, "할아버지"에서 "아버지"로 표시됩니다.

예를 들어 기본적으로 "나" 기준에서 할머니는 "할머니(2촌)"로 표시되지만, 할머니를 한 번 탭하면 할머니 기준으로 모든 호칭이 재계산됩니다. 그러면 나는 "손자(2촌)"로 표시되고, 아버지는 "아들(1촌)"로 바뀝니다.

원래 기준("나")으로 돌아가려면 헤더 우측의 "기준: OOO" 영역을 탭하면 됩니다.

5단계 — 관계 검색

화면 하단의 관계검색 탭을 누르면 두 사람을 선택해 관계를 조회할 수 있습니다.

  • A가 B를 부를 때의 호칭
  • B가 A를 부를 때의 호칭 (역방향)
  • 정확한 촌수
  • 두 사람을 잇는 경로(가계도 상에서 하이라이트 표시)

특히 "사촌의 아내"나 "배우자의 외삼촌"처럼 헷갈리는 관계도 정확히 알려드립니다.

6단계 — 호칭 퀴즈

하단 탭바의 🎯 호칭퀴즈 탭을 누르면 가계도 속 가족의 호칭을 4지선다로 맞출 수 있습니다.

  • 기준 인물 기준으로 가족 호칭 문제 출제
  • 총 10문제, 정답 여부 즉시 확인
  • 최종 점수와 정답률 표시

💡 팁: 가족이 많을수록 더 다양한 문제가 출제됩니다. 명절 전에 미리 연습해보세요!

7단계 — 생일 알림

촌맵은 가족의 생일을 기억하고 알려드립니다.

생일 입력

가족 정보 수정 화면에서 생일(월-일) 항목에 03-15 형식으로 입력하세요. 출생년도와 별개로 매년 반복 알림됩니다.

인앱 배너

오늘 또는 내일 생일인 가족이 있으면 가계도 화면 상단에 자동으로 배너가 표시됩니다.

  • 🎂 오늘: 홍길동(아버지) 생일
  • 🎁 내일: 김영희(어머니) 생일

시스템 알림 (Android)

설정 탭 → 🎂 생일 알림 토글을 켜면 앱 실행 시 안드로이드 알림 바에 생일 알림이 표시됩니다.

  1. 설정 탭(⚙️) 진입
  2. "🎂 생일 알림" 토글을 켬
  3. 알림 권한 허용
  4. 이후 앱 실행 시 생일 당일·전날 자동 알림

8단계 — 데이터 백업과 복원

모든 데이터는 브라우저의 로컬 저장소에 저장되므로, 기기를 바꾸거나 브라우저를 초기화하면 데이터가 사라질 수 있습니다. 정기적으로 백업하시기를 권장합니다.

내보내기

설정 > JSON으로 내보내기를 선택하면 현재 가계도가 chonmap-backup.json 파일로 저장됩니다.

가져오기

설정 > JSON 가져오기에서 저장해 둔 파일을 선택하면 그 상태로 복원됩니다. 기존 데이터는 덮어쓰기 되므로 주의하세요.

이미지로 저장

설정 > 이미지로 저장을 선택하면 가계도를 PNG 파일로 내보낼 수 있습니다. 가족 단체 채팅방에 공유하거나 프린트하기에 좋습니다.

PDF 족보 내보내기

설정 > 📄 PDF 족보 내보내기를 선택하면 가족 전체 목록을 표 형식으로 정리한 새 창이 열리고, 인쇄 대화상자가 자동으로 실행됩니다. "PDF로 저장"을 선택하면 족보 파일을 저장할 수 있습니다.

  • 이름·성별·호칭·촌수·출생년도·사망년도·메모 포함
  • 기준 인물 기준으로 호칭 자동 계산

앱 공유하기

설정 > 🔗 앱 공유하기를 누르면 촌맵 링크를 카카오톡·문자·SNS 등으로 바로 공유할 수 있습니다. 기기가 Web Share API를 지원하지 않는 경우에는 링크가 클립보드에 복사됩니다.

9단계 — 앱 평가하기

촌맵을 5회 이상 실행하고 가족을 5명 이상 등록하면 앱 평가 다이얼로그가 나타납니다.

  • 지금 평가하기: Google Play 스토어로 이동해 별점을 남겨주세요
  • 나중에: 7일 후 다시 표시됩니다
  • 다시 보지 않기: 더 이상 표시되지 않습니다

💛 개발자 한마디: 별점 5점과 리뷰 한 줄이 개발자에게 큰 힘이 됩니다. 촌맵이 마음에 드셨다면 꼭 남겨주세요!

10단계 — 가계도 화면 크게 보기

가족이 많아질수록 하단 가족 목록이 가계도 공간을 차지합니다. 가계도를 더 크게 보고 싶을 때는 목록을 숨길 수 있습니다.

  • 하단 가족 목록 우상단 × 버튼을 누르면 목록이 숨겨지고 가계도가 더 넓어집니다.
  • 다시 보려면 하단에 나타나는 👥 가족 목록 보기 버튼을 누르세요.
  • 선택은 저장되어 다음 실행 시에도 유지됩니다.

11단계 — 팬·줌으로 큰 가계도 탐색

  • 드래그: 마우스나 손가락으로 끌어 화면 이동
  • 마우스 휠: 커서 위치 기준으로 확대/축소
  • 핀치: 두 손가락으로 확대/축소
  • 전체보기 버튼: 가계도 전체가 한 화면에 들어오도록 자동 조정

12단계 — 가족 순서 바꾸기 (드래그)

형제자매나 자녀의 배치 순서를 바꾸고 싶을 때 드래그로 좌우 이동할 수 있습니다.

  1. 순서를 바꾸고 싶은 사람을 0.5초 꾹 누릅니다 (진동 피드백)
  2. 노드가 반투명해지며 드래그 모드 시작 — 좌우로 끌면 황금색 점(●)이 놓일 위치를 안내합니다
  3. 손을 떼면 새로운 순서로 즉시 반영됩니다

💡 팁: 실수했다면 Ctrl+Z(되돌리기)로 바로 복구할 수 있습니다. 부모가 없는 단독 인물은 드래그 대상이 아닙니다.

자주 묻는 질문

Q. 배우자를 여러 명 추가할 수 있나요?

A. 한 사람당 배우자는 1인으로 제한되어 있습니다. 재혼·이혼 등 복잡한 관계는 현재 지원하지 않습니다.

Q. 가계도를 가족과 공유할 수 있나요?

A. JSON 파일이나 이미지로 내보내 메신저로 공유하실 수 있습니다. 서버 기반 공유 기능은 현재 제공되지 않습니다.

Q. 데이터가 갑자기 사라졌어요.

A. 브라우저의 사이트 데이터 삭제, 시크릿 모드 사용, 저장소 용량 초과 등이 원인일 수 있습니다. 정기적으로 JSON 백업을 권장합니다.
